## Notas para continuar

- **EPUB es un ZIP**: El WebView intenta cargar `file:///path/book.epub` directamente, pero EPUB es un archivo ZIP. Hay que extraer los HTML del EPUB y servirlos al WebView, o usar un ContentProvider. Este sera probablemente el proximo bug a resolver.
- **EpubParser ya extrae contenido**: `extractTextContent()` parsea los HTML dentro del ZIP y devuelve texto plano. Se puede usar para TTS. Para lectura visual, hay que extraer los HTML a un directorio temporal y cargarlos en el WebView.
- **Debug con ADB**: Limpiar logcat (`adb logcat -c`), reproducir crash, capturar (`adb logcat -d | grep ebookreader`).
- **Codex CLI** disponible en la maquina (`codex exec`) para delegar tareas.
- **Build**: `JAVA_HOME="C:/Program Files/Android/Android Studio/jbr"`, `ANDROID_HOME="$LOCALAPPDATA/Android/Sdk"`
- **Instalar**: `adb install -r app/build/outputs/apk/debug/app-debug.apk`
