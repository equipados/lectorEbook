## Contexto

App Android nativa (Kotlin + Jetpack Compose) para leer ebooks EPUB y PDF con sintesis de voz (TTS local offline y Google Cloud TTS online). MVP completo: compila, APK generado e instalado en movil del usuario (Android 15). UI en español.

## Notas para continuar

- El EpubParser fue reescrito para NO usar Readium Streamer (API 3.x incompatible). Usa ZipFile nativo.
- feature:reader mantiene dependencia de readium-navigator en build.gradle.kts pero el EpubReaderView usa WebView placeholder.
- Para compilar: `JAVA_HOME="C:/Program Files/Android/Android Studio/jbr"` y `ANDROID_HOME="$LOCALAPPDATA/Android/Sdk"`
- El movil necesita "Instalar via USB" activado ademas de Depuracion USB.
- Se uso Codex CLI (`codex exec`) para 4 de las 15 tareas originales.
